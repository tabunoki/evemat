EVE MAT
===============================================================================
EVE Online manufacture assistance tool


About
-------------------------------------------------------------------------------
EVE MAT は EVE Online のゲーム内生産を援助するツールです。
IGB から利用できるため、プレイしながらツールを利用することが可能です。



Development
-------------------------------------------------------------------------------
EVE MAT は Eclipse での開発を想定しています。
